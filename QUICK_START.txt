QUICK START GUIDE
Online Food Ordering System - MongoDB Edition
===========================================

Follow these simple steps to get your application running:

STEP 1: Install MongoDB Driver (2 minutes)
-------------------------------------------
Open your terminal/command prompt in the project folder and run:

    composer install

Wait for it to complete. You should see "vendor" folder created.


STEP 2: Start MongoDB (1 minute)
---------------------------------
1. Open MongoDB Compass application
2. Click "New Connection" or use existing connection
3. Connection string: mongodb://localhost:27017
4. Click "Connect"


STEP 3: Create Database (2 minutes)
------------------------------------
In MongoDB Compass:
1. Click "+" button or "Create Database"
2. Database Name: online_food_ordering_system
3. Collection Name: admins
4. Click "Create Database"

Now create 3 more collections:
- Click your database name
- Click "+" or "Create Collection"
- Create: users
- Create: foods  
- Create: orders
